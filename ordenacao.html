Selection Sort


In-place algorithm


  1. Encontre o menor elemento no array.
  2. Troque-o com o elemento da primeira posição.
  3. Encontre o segundo menor elemento.
  4. Troque-o com o elemento da segunda posição.
  5. Continue até o array estar ordenado.

Selection Sort


Abaixo a implementação da Ordenação por Seleção, ou Selection Sort, na linguagem de programação C.

In-place algorithm 

void selectionSort(int V[], int n)
{
    int i, j, m;
    for (i = 0; i < n - 1; i++) {
        m = i;
        for (j = i + 1; j < n; j++)
            if (V[j] < V[m])
                m = j;
        swap(&V[i], &V[m]);
    }
}

Bubble Sort


Abaixo a implementação da Ordenação por Flutuação, ou Bubble Sort, na linguagem de programação C.

In-place algorithm 

void bubbleSort(int V[], int n)
{
    int i, j;
    for (i = 0; i < n; i++)
        for (j = 0; j < n - 1; j++)
            if(V[j] > V[j + 1])
                swap(&V[j], &V[j + 1]);
}
